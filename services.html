# Gemini Coach Website Complete website ready for GitHub Pages hosting. ## Files included - `index.html` — Homepage - `about.html` — About Parker - `services.html` — Services & Pricing - `podcast.html` — Podcast & Features - `blog.html` — Blog (ready for posts) - `refund-policy.html` — Refund Policy - `privacy.html` — Privacy Policy - `thank-you.html` — Form submission confirmation - `images/` — All photos and logo files ## How to publish on GitHub Pages 1. Go to github.com and create a free account if you don't have one 2. Click the "+" button top right and select "New repository" 3. Name it exactly: `geminicoach` (or your preferred name) 4. Leave it set to Public 5. Click "Create repository" 6. Click "uploading an existing file" link on the next screen 7. Drag the entire contents of this folder into the upload window 8. Click "Commit changes" 9. Go to Settings > Pages 10. Under "Source" select "Deploy from a branch" 11. Select "main" branch and "/ (root)" folder 12. Click Save 13. Your site will be live at: `https://yourusername.github.io/geminicoach` ## Adding your hero video 1. Record or export your training video (15-30 seconds, MP4 format) 2. Name the file exactly: `hero-video.mp4` 3. Upload it to the `images/` folder in your GitHub repo 4. Open `index.html` and find this comment: `` 5. The video will automatically replace the photo background ## Adding blog posts Each blog post is a new HTML file. Copy this starter template, save it as `blog-post-1.html` and link to it from `blog.html`. ## Custom domain (optional) If you want geminicoach.com to point to this site instead of WordPress: 1. Go to Settings > Pages in your GitHub repo 2. Enter your custom domain 3. Update your domain DNS records to point to GitHub Pages IPs 4.
